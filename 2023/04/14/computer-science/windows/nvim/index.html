別問哪麼多，因為我是在 window 裡面裝的當然…。
對我來說 neovim 絕對比 vscode 香太多了，撇開套件非堂多之外。能用純鍵盤打字是吸引的。

# 什麼是 vim, 以及什麼是 neovim

vim 是款編輯器，對於普通的使用者來說有點類似 vscode, 但他是純命令控制的方式。
而 neovim 是基於 vim 的部分改進，而兩個是不同的作者在維護的。
